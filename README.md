# Image Processor

Учебный проект на C++ для обработки изображений в формате BMP.  
Программа читает входное изображение, последовательно применяет к нему набор фильтров и сохраняет результат в новый файл.

## Возможности

Проект поддерживает следующие фильтры:

- crop — обрезка изображения до заданных размеров
- gs — перевод изображения в оттенки серого
- sharp — повышение резкости
- edge — выделение границ
- blur — гауссово размытие

Фильтры применяются в том порядке, в котором переданы в командной строке.

---

## Формат запуска

```bash
./image_processor <input.bmp> <output.bmp> [filters...]
```

## Поддерживаемые фильтры
## Crop
```bash
-crop <width> <height>
```

## GrayScale
```bash
-gs
```

## Sharpen
```bash
-sharp
```

## Edge Detection
-edge <threshold>

## Gaussian Blur
```bash
-blur <sigma>
```

## Сборка
```bash
mkdir build
cd build
cmake ..
make
```
